# 社群登入設定（Facebook / LINE）

程式碼已經接好了。要讓 Facebook、LINE 按鈕真的能登入,你只要去各平台申請一組金鑰,填進 `.env` 就會自動啟用。**沒填的話,按鈕點下去會顯示「尚未設定」的提示,不會壞掉。**

> 訪客快速登入、Google 登入不需要做這份設定,已經可以用。

---

## 一、Facebook 登入

1. 到 **https://developers.facebook.com** → 用你的 FB 帳號登入 → 右上「我的應用程式 My Apps」→「建立應用程式 Create App」。
2. 用途選「**驗證與帳號要求(Authenticate and request data from users）/ Consumer**」之類 → 建立。
3. 進到應用程式後,左側加入「**Facebook 登入 Facebook Login**」產品 → 設定 Settings。
4. 在「**有效的 OAuth 重新導向 URI（Valid OAuth Redirect URIs）**」填入:
   ```
   http://localhost:5001/login/facebook/callback
   ```
   > ⚠️ Facebook 通常不接受 `127.0.0.1`,請改用網址列輸入 **http://localhost:5001** 來測試登入。
5. 到「應用程式設定 → 基本資料」拿到 **應用程式編號 App ID** 和 **應用程式密鑰 App Secret**。
6. 填進 `.env`:
   ```
   FACEBOOK_CLIENT_ID=你的App_ID
   FACEBOOK_CLIENT_SECRET=你的App_Secret
   ```

---

## 二、LINE 登入

1. 到 **https://developers.line.biz** → 用 LINE 帳號登入。
2. 建立一個 **Provider**(供應商,名稱隨意)→ 在裡面「Create a new channel」選 **LINE Login**。
3. 填基本資料建立後,到該 channel 的「**LINE Login**」分頁:
   - 「**Callback URL**」填入:
     ```
     http://127.0.0.1:5001/login/line/callback
     ```
   - 若要拿到 email,到「OpenID Connect」區塊申請 email 權限(選用)。
4. 到「**Basic settings**」分頁,拿到 **Channel ID** 和 **Channel secret**。
5. 填進 `.env`:
   ```
   LINE_CLIENT_ID=你的Channel_ID
   LINE_CLIENT_SECRET=你的Channel_secret
   ```

---

## 三、填完之後

1. 存檔 `.env`。
2. 回終端機按 `Ctrl + C` 停掉網站,再 `python app.py` 重新啟動。
3. 登入頁的 Facebook / LINE 按鈕就會真的能登入了。

> 小提醒:三種登入(Google / FB / LINE)的「重新導向 / Callback URL」網址結尾各不同,別填錯:
> - Google → `http://127.0.0.1:5001/login/callback`
> - Facebook → `http://localhost:5001/login/facebook/callback`
> - LINE → `http://127.0.0.1:5001/login/line/callback`
